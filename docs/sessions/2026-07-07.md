### Bloqueador da revisão
- **B1**: Standalone renderizava o painel MLC inline, sem os controles do limiter. Corrigido adicionando Enable toggle + Ceiling/Release knobs inline com valores absolutos.

### O que foi feito

#### Módulo DSP: `src/core/dsp/limiter.rs` (novo, 136 linhas)
```
PeakLimiter {
    ceiling: f32,        // teto linear (ex: 10^(-1/20) ≈ 0.891)
    envelope: f32,       // estado do envelope de pico
    release_coeff: f32,  // coeficiente de release por sample
}
```

**Algoritmo:**
1. `abs_sample = |sample|`
2. Se `abs_sample > envelope`: ataque instantâneo → `envelope = abs_sample`
3. Senão: release exponencial → `envelope += (abs_sample - envelope) × coeff`
4. Se `envelope > ceiling`: gain reduction → `sample ×= ceiling / envelope`
5. Retorna sample (garantia: `|sample| ≤ ceiling`)

**Propriedades:**
- Zero-latência (sem lookahead, sem PDC)
- Ataque sample-accurate — nenhum pico escapa
- Release exponencial suave — sem pumping audível
- Puro Rust — sem FFI, sem dependências externas

#### Parâmetros (`src/core/state/plugin_params.rs`)

| Parâmetro | ID | Range | Default | Suavização |
|---|---|---|---|---|
| Limiter Enable | `lim_en` | Bool | true | — |
| Limiter Ceiling | `lim_ceil` | -12 a 0 dB | -1.0 dB | Linear 10ms |
| Limiter Release | `lim_rel` | 10 a 500 ms | 50 ms | Linear 50ms |

#### Posição na pipeline (`src/lib.rs`)
```
Input → EQ → Pre-EQ Conv → Amp (MLC/Neural) → Cabinet IR → Master Gain (+30 dB)
    → 🆕 BRICKWALL LIMITER → NaN Sanitization → Output
```

Inserido **após** Master Gain (+ lab pipeline) e **antes** da sanitização NaN, capturando clipping de TODAS as fontes combinadas.

#### UI (`src/core/ui/mlc_zero_v_panel.rs` + standalone inline)
```
┌─ LIMITER ──────────────────────────┐
│ [✓] Enable   ┌──────┐  ┌──────┐   │
│              │ Ceil │  │  Rel │   │
│              │ -1.0 │  │50 ms │   │
│              │  dB  │  │      │   │
│              └──────┘  └──────┘   │
└───────────────────────────────────┘
```

#### Garantia matemática
```
Pior caso: MLC gain 0dB × master 0dB × cab × master +30dB = sinal 31.62×
Com ceiling -1.0 dB (0.891 linear):
    |saída| ≤ 0.891 < 1.0  → NUNCA CLIPA
```
